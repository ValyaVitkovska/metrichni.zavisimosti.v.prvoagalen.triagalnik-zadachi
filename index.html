Метрични зависимости и формули

▸ Означения
a=BC, b=AC, c=AB hc=CH  (височина) a1=HB, b1=AH  (проекции на катетите)
▸ Метрични зависимости
1. a2 = c · a1 2. b2 = c · b1 3. hc2 = a1 · b1 4. a · b = c · hc 5. a2 + b2 = c2 6. c = a1 + b1
▸ Лице, периметър, окръжности
S = a·b2 = c·hc2 P = a + b + c R = c2 r = a + b − c2
